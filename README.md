# LS45 Automotriz - Diseño Web

Proyecto desarrollado para la asignatura **Diseño Web**, correspondiente al sitio web de la automotora **LS45 Automotriz**.

## Descripción del proyecto

Este proyecto consiste en el desarrollo de una página web para **LS45 Automotriz**, una empresa dedicada a la compra, venta y asesoría automotriz.  
El sitio tiene como objetivo presentar los servicios de la automotora, mostrar vehículos disponibles, entregar información de contacto y ubicación, además de dar a conocer la misión y visión de la empresa.

La página fue construida utilizando tecnologías web básicas, aplicando estructura semántica con HTML5 y estilos CSS para mejorar la presentación visual y la experiencia de navegación del usuario.

## Secciones del sitio

El sitio web incluye las siguientes secciones principales:

- **Inicio:** presentación general de la empresa.
- **Servicios:** información sobre los servicios ofrecidos.
- **Stock:** vehículos disponibles para la venta.
- **Nuestros clientes:** imágenes de entregas realizadas a clientes.
- **Contacto y ubicación:** dirección, redes sociales y canales de comunicación.
- **Sobre nosotros:** misión y visión de la empresa.
